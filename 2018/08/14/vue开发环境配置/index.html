vue开发环境配置

2018-08-14 • 标签: vue

##自己动手搭建Vue开发环境

$ npm install -g cnpm –registry=https://registry.npm.taobao.org

基于node.js,利用淘宝npm镜像安装相关依赖

$ cnpm install vue-cli -g -S 安装脚手架

$ vue / vue list (查看可安装的模板)

$ vue init webpack report(项目名称)

（init的选项）

? Project name（回车）

? Project description （回车）

? Author （回车）

? Use ESLint to lint your code? Yes

? Pick an ESLint preset Standard（回车）

? Setup unit tests with Karma + Mocha? No

? Setup e2e tests with Nightwatch? No

$ cd report 新建的vue项目

$ cnpm i element-ui -S

$ cnpm i axios -S

$ cnpm i mockjs -S

$ npm run dev
